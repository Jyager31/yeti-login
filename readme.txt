A fun, animated yeti character for the WordPress login page with dark styling and GSAP-powered eye tracking.

== Description ==

Yeti Login replaces the default WordPress login page with a sleek dark design featuring an animated yeti character. The yeti watches as you type your username and covers its eyes when you enter your password.

**Features:**

* Animated yeti SVG that tracks cursor position in the username field
* Yeti covers eyes during password entry
* Finger-spread animation on password reveal toggle
* Natural blinking animation
* Dark, modern login page styling with glassmorphism effects
* Customizable logo, logo link, and background image
* Works with any WordPress theme
* No frontend performance impact — assets only load on wp-login.php

**Credits:**

Yeti character animation originally by [Darin Senneff](https://darinsenneff.com). Plugin by [DevQ](https://thedevq.com/).

== Installation ==

1. Upload the `yeti-login` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Optionally customize settings at Settings > Yeti Login

== Frequently Asked Questions ==

= Does this affect my site's frontend performance? =

No. All CSS and JavaScript are only loaded on the WordPress login page (wp-login.php).

= Can I use my own logo? =

Yes. Go to Settings > Yeti Login and upload a logo image using the media uploader.

= Can I change the background? =

Yes. The plugin ships with a dark brick background, but you can upload your own at Settings > Yeti Login.

= Does this work with any theme? =

Yes. Yeti Login is a standalone plugin with no theme dependencies.
